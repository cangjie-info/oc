DB ‘cangjie’ created.

DB table graph_components created

One row represents the presence of one graph as a component of another. Simple graphs do not have an entry in this table. A component C of a component B of a graph A will only be listed as a component of B, not of A. To the extent possible, this table will be populated by existing data.

Table should have no duplicate rows. Table should represent historical relationships, not surface similarities in the contemporary orthography.
